# WallpaperCompose 🌄

Wallpaper Compose, an Android app that combines modern design with a vast collection of high-quality wallpapers. 🚀

## Features:

- Browse a curated collection of breathtaking wallpapers.
- Easily search for wallpapers by keywords.
- Immerse yourself in full-screen viewing mode.
- Show your appreciation by liking your favorite wallpapers.
- Memory and Network Cache Integrated.

Experience a seamless and responsive UI built with Jetpack Compose. 🎨

## Tech Stack 🛠️:

- [Jetpack Compose](https://developer.android.com/jetpack/compose): The future of Android UI development.
- [Dagger Hilt](https://dagger.dev/hilt/): For dependency injection.
- [Retrofit](https://square.github.io/retrofit/): A powerful HTTP client for network requests.
- [HttpClient](https://developer.android.com/reference/java/net/HttpClient): Android's built-in HTTP client.
- [Pexels API](https://www.pexels.com/api/documentation/): Access to a treasure trove of high-quality images.
- [MVVM](https://developer.android.com/jetpack/guide): A clean architecture pattern for maintainable code. 💡
- [Coil](https://coil-kt.github.io/): A image fetching and displaying library. 💡
